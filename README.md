# Brazil-E-Commerce

<h2>Description</h2>
This project, completed for a Database Management class in Spring 2024, involves the analysis of data from a Brazilian e-commerce company to derive actionable insights and recommendations. Using MySQL for data management and Tableau for data visualization, this project aims to provide a comprehensive understanding of the company’s operational metrics, customer behaviors, and sales trends.
<br />


<h2>Languages and Utilities Used</h2>

- <b>SQL</b> 
- <b>Tableau</b>

<h2>Environments Used </h2>

- <b>Macbook Air </b> (21H2)
