# Brazil-E-Commerce

<h2>Description</h2>
This project, completed for a Database Management class in Spring 2024, involves the analysis of data from a Brazilian e-commerce company to derive actionable insights and recommendations. Using MySQL for data management and Tableau for data visualization, this project aims to provide a comprehensive understanding of the companyâ€™s operational metrics, customer behaviors, and sales trends.
